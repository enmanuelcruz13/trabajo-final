🚀 INICIO RÁPIDO:

1. Activar entorno virtual:
   .\.venv\Scripts\activate

2. Iniciar servidor:
   python manage_new.py runserver

3. Acceder a:
   - Catálogo: http://127.0.0.1:8000/
   - Admin: http://127.0.0.1:8000/admin/

📝 COMANDOS ÚTILES:

Migrar base de datos:
  python manage_new.py migrate

Crear usuario admin:
  python manage_new.py createsuperuser

Cargar datos de ejemplo:
  python manage_new.py seed

Hacer migraciones:
  python manage_new.py makemigrations

Con Make (si está instalado):
  make install
  make migrate
  make createsuperuser
  make runserver

❓ PROBLEMAS?

1. Lee: GUIA_MIGRACION.md (Troubleshooting)
2. Verifica: python manage_new.py check
3. Asegúrate: Estás en la carpeta correcta (donde está manage_new.py)
4. Confirma: .venv está activado
