Resumen del Proyecto

30 películas de 6 géneros, todos los trailers convertidos a español (latino y de España), verificación mediante oEmbed, y despliegue en Render con Supabase PostgreSQL.

Proceso Técnico

🔍

Búsqueda manual

Búsqueda en YouTube de trailers oficiales en "Español Latino" y "Español (España)" para cada película.

✅

Verificación oEmbed

Cada ID verificado con la API oEmbed de YouTube para confirmar que existe, es público y embeddable.

🔄

Re-seed en Render

Commit sin flag --if-empty para forzar la re-población, seguido de commit de restauración.

🐘

Supabase PostgreSQL

Base de datos PostgreSQL gestionada en Supabase, conectada via dj_database_url.
